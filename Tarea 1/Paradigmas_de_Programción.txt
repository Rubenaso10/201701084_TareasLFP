Lenguajes de programación Imperativos:
Defininen un programa como un conjunto de instrucciones u ordenes que se le da, contestando la pregunta ¿Cómo se hace?.
Algunos ejemplos de lenguajes de programación imperativo sería: C, C++, Java, Python.
Este sería el código de un programa que determna el factorial de un número:

procedimiento factorial (entero n) 
{
	entero resultado = 1;


	mientras (n>0) 

	{
	resultado = resultado *n ;
	n= n-1;

	}
	devuelve resultado;
}


Lenguajes de programación declarativos:

declarativo. Ejemplo:
